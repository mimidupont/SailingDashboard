# Tamarin — Carnet de Maintenance
### Trisbald 36 · Voile n° FRA 7284

Application de suivi de maintenance pour le voilier **Tamarin**.

## Démarrage local

```bash
npm install
npm start
```

Ouvre automatiquement sur **http://localhost:3000**

## Déploiement

Ce projet est déployé automatiquement sur **Vercel** à chaque push sur `main`.
